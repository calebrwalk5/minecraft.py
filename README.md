# minecraft.py
INSTALLATION:
You will need to install Pyglet with these terminal commands

pip install pyglet
git clone https://github.com/calebrwalk5/minecraft.py

After that run minecraft.py in Python

CONTROLS:
WASD to move around
Mouse to look around
Tab to fly
Esc to release cursor

PLACING BLOCKS:
1. Wood
2. Grass
3. Sand
4. Brick
5. Daimonds
6. Rocks
